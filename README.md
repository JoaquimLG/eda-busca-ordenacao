# EDA — Busca e Ordenação em Java

Um repositório com **implementações de algoritmos de busca e ordenação em Java**, aplicados a objetos, com foco no estudo de **Estruturas de Dados e Algoritmos**.

---

## Descrição

Este projeto foi desenvolvido com o objetivo de praticar e consolidar conceitos fundamentais de **Estruturas de Dados**, por meio da implementação de algoritmos clássicos de **busca** e **ordenação** em Java.

O código foi organizado de forma a facilitar o entendimento, a manutenção e a evolução do projeto, separando responsabilidades em pacotes bem definidos.

---

## Organização do Projeto

O programa está dividido nos seguintes pacotes principais:

- **searching**  
  Contém os algoritmos responsáveis pela busca de elementos em estruturas de dados.

- **sorting**  
  Reúne as implementações dos algoritmos de ordenação, permitindo analisar diferentes estratégias e comportamentos.

- **utils**  
  Contém classes utilitárias de apoio ao funcionamento do sistema, como geração de dados e funções auxiliares.

Essa separação ajuda a manter o código mais limpo, modular e fácil de compreender.

---

## Algoritmos Implementados

### Algoritmos de Busca
- Busca Linear (Iterativa e Recursiva)
- Busca Binária (Iterativa e Recursiva)
- Busca Linear Iterativa Duas Pontas
  
### Algoritmos de Ordenação
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort  
(*Verifique o pacote `sorting` para a lista completa e suas variações.*)

## Pré-requisitos

Para executar o projeto localmente, é necessário:

- **Java JDK 8 ou superior**
- Uma IDE Java, como:
  - IntelliJ IDEA
  - Eclipse
  - VS Code com extensão Java

---

## Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/JoaquimLG/eda-busca-ordenacao.git
   ```

2. Abra o projeto na sua IDE de preferência.

3. Compile o projeto.

4. Execute a classe `Main`, responsável por controlar o fluxo do programa e permitir a execução dos algoritmos.
